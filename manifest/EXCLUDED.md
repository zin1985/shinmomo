# Excluded files policy

The following must not be committed in this handoff package:

```text
*.smc
*.sfc
*.bin
*.zip
*.7z
*.state
*.sav
savestate/
vram_raw/
oam_raw/
cgram_raw/
raw_dumps/
```

Reason:

```text
ROM binaries and raw copyrighted assets must stay outside the repository handoff.
This package preserves analysis, schemas, scripts, and reconstruction notes only.
```
