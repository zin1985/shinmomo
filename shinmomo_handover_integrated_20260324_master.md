## 1. 現在の全体像

### 1-1. 店/施設/分岐/UI 系の本線
- `0x41A10..` は **条件分岐表本命**。
- `0x41A10 -> 0x39850 / 0x39993` は **target-side の施設/イベント blob 系**。
- `0x41D38` は second-table の record ではなく **独立 8byte header**。
- `0x41D20` の 10byte 塊は **first/second table 間の remap / skip metadata** 候補。
- `0x39847` A系 / `0x39916` B系は **23件×9byte の兄弟レコード列**。
- `0x39850` は A#1 先頭。
- `0x39993` は clean record start ではなく **B系途中ラベル / 再開境界**。

### 1-2. bank82 local rule / membership engine 系
- `1D8A -> BDE7 -> C379/C931` は **bank82 内の local rule / membership engine**。
- `19D5` は施設種別そのものではなく **choice layer**。
- `ledger (1B15/1B25/1B35) -> bucket (BCA5/BCE1/BD06) -> choice (1D77/1D87)` の中間層が見えている。
- `398xx/399xx` blob と `C379 builder` 側は **別 subsystem**。

### 1-3. 文字・会話系の本線
- 文字表示主線は **bank89**。
- 大枠は **descriptor -> raw ID -> token 展開**。
- 重要節点:
  - `89:9D4D`
  - `89:9D91`
  - `89:9E10`
  - `89:9DE5`
  - `89:9EC7`
  - `89:A5AF/A5C8`
- `89:9E10` の分類は次で読むのが自然。
  - `00` = 終端
  - `01..4F` = 制御
  - `50..FF` = literal raw ID
- `0x70000` 台は **本文直置きより、C7:0000 起点 descriptor / pointer / page header / small dictionary 側**。
- `0x30000` 台は **script / event 本体寄り**。

### 1-4. いま未完成の本丸
- `0x41A10` reader 本体は未発見。
- script 帯と descriptor 帯を結ぶ最終ルートは未完成。
- glyph 最終変換表と全文自動抽出は未完成。

## 3. token クラスと文字層の整理

### 3-1. 基本制御
- `00` = 終端
- `01` = 改行 / 行区切り候補
- `50` = 行内 delimiter / 空白相当候補
- `5B` = packet 終端候補
- `7D .. 7E` = quoted speech / 発話 packet 候補
- `04 ... 03` = 小 macro / structured control / page-head template

### 3-2. 文字クラス
- `18 xx`
  - **漢字1字、または漢字語幹** が多い
- `19 xx`
  - **熟語・短句・敬称込み圧縮形・定型句**
- `1A xx`
  - **短句・場面依存の定型句・文脈語彙**
- `50..FF`
  - **ひらがな / カタカナ / 擬音 / 記号 literal 側**

### 3-3. 現在の実用ルール
新桃は **1文字ずつの単純表ではなく、漢字辞書 + かな literal + 熟語辞書 + ページ頭テンプレ** の混成構造とみるのが自然。
