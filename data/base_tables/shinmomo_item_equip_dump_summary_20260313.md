
# 新桃太郎伝説 道具表・装備表ダンプ更新（2026-03-13）

## 今回の新規整理

### 道具表
- 開始: `0x442BF`
- レコード長: **11 bytes**
- レコード数: **164**
- ID対応: **01..A4**
- 最終レコード開始: `0x449C0`
- 最終レコード終端: `0x449CA`
- その後 `0x449CB..0x449CF` に **5バイトのゼロ埋め**
- Beta版と製品版Jで **差分なし**

実用フィールド:
- `+00..02` = 3バイトの種別/挙動フラグ
- `+03..04` = 買値
- `+05..06` = 売値
- `+07..08` = 効果量/値
- `+09` = 追加引数候補（現状は `打ち出のこづち` と `焼きイモ` でのみ `05`）
- `+0A` = 追加引数候補（現状は全件 `00`）

### 装備表
- 開始: `0x449D0`
- レコード長: **16 bytes**
- 実用ダンプ範囲: **ID 01..E6**
- 最終レコード開始候補: `0x45820`
- 最終レコード終端候補: `0x4582F`
- Beta版と製品版Jで **差分なし**

実用フィールド:
- `+00..02` = 3バイトの種別/特殊フラグ
- `+03..04` = 買値
- `+05..06` = 売値
- `+0E` = 装備スロット
  - `00` 武器
  - `01` 頭
  - `02` 胴
  - `03` 足
- `+0F` = 能力上昇値（攻撃/防御などの主値）
