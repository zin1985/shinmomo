## 3. 重要な未調査ポイント

- 41A10.. 条件表を実際に走査して target を抜く本体コード
- 31C68 への直結ルート
- 所持金RAM本体と価格比較・購入可否判定・減算ルート
- 442BF / 449D0 を直接または間接に参照する価格読取コード
- 81:8D87 本体と戻り 09/0A/0C/0D の意味
- 7E:718B / ($9C) のさらに先の最終 tilemap / DMA 経路

## 4. 今回の読み方の注意

- `0x41AC0` は単独で見るより **`0x41A10..` の中腹** として扱うのが安全。
- `7E:7189..718B` は今のところ **描画終点ではなく、bank89 の表示VMが使う staging / operand stack** と見るのが最もしっくり来る。
- `7E:200x / 280x / 300x` は **tilemap/DMA直バッファより、画面用中間構造体** の可能性が高い。
- `0x4F` は会話分岐命令というより **UI/施設画面の field 抽出命令** と捉えると整理しやすい。

## 5. 補足

本ファイルは、既存の引継ぎメモと今回までの追跡結果を統合して作成した“見取り図”です。
厳密な意味での『ROMの全アドレスを全部表示』ではありません。2MB 全域の逐一列挙は実用性が低く、ノイズが大きすぎるため、
**解析に意味のある単位で全部見える化** する方針を採っています。
