# 新桃太郎伝説 Web攻略情報照合による未解決文字・施設UI解析メモ（2026-04-24）

## 目的
Web上の攻略情報で確認できる実ゲーム仕様を、ROM内の未解決テキスト・施設UIメッセージ帯に照合し、Goal 5 / Goal 9 / Goal 8 の進捗を伸ばす。

## Web情報から使える確認点
- 人気度は 81以上で茶店・宿屋側の優遇が発生し、0〜29で価格ペナルティやお供命令不可が発生する。
- 別攻略情報では「人気度80以上」と表現されることがあるが、ROM文面では「80をこえた」と読めるため、実装表示上は「80超え = 81以上」と整理するのが自然。
- といちやは道具・金の預かり所で、といちや加入条件として「預けたお金が30001両以上 / 30000両で雇用」が確認できる。
- 怨みの洞窟は「嫌いな人の名前を3人入力」するイベントであり、ROM側の「いちばんきらい」「2番目」「最後」プロンプトと合う。

## ROM照合で更新できた箇所

### 1. 人気度80超え表示
- ROM: `0x083420..0x083458`
- 既読断片:
  - `人気度が 80をこえたので`
  - `半分の値段で`
  - `宿屋に とまれるようになった!`
- 解釈:
  - 人気度が 81以上になった時の宿屋割引・解禁案内。
  - 攻略情報の「81〜」表記とROMの「80をこえた」が一致。
- Goal影響:
  - Goal 5: 人気度条件による施設UI表示として確定度上昇。
  - Goal 9: システムメッセージ仕様に条件文を追加。

### 2. 人気度30未満表示
- ROM: `0x0830DE..0x08310F`
- 既読断片:
  - `人気度が 30をわったので`
  - `いつもの 2ばいに`
  - `ちりょうひを はらうはめに なってしまった!`
- 解釈:
  - 人気度 0〜29 の治療費2倍ペナルティ文。
  - Web情報では茶店・宿屋のみとする記述もあるが、別情報では医者も値上がり対象。ROM文から医者/治療費2倍はかなり強い。
- Goal影響:
  - Goal 5: 医者UI料金補正が人気度ペナルティに接続。
  - Goal 9: 人気度低下システム文として確定度上昇。

### 3. といちや / 十一屋
- ROM: `0x082E30..0x082E50`, `0x082FE0..0x083035`
- 既読断片:
  - `十一屋`
  - `以上 ためていると`
  - `十一屋グループのしくみを説明しまひょ`
  - `65000両まで`
- 解釈:
  - 表記は「十一屋」と読めるが、実ゲーム仕様上は「といちや」系。
  - 預け金しきい値・グループ説明・預入上限説明が同じ帯にある。
  - 過去の `0x39839` 近辺の `0x7530` = 30000候補は、といちや雇用/預け金しきい値として再検討価値が高い。
- Goal影響:
  - Goal 5: 預かり所/銀行UIの分類が前進。
  - Goal 8: 30000両条件 blob の意味候補が強化。

### 4. 怨みの洞窟
- ROM: `0x09BE1D..0x09BF5A`
- 既読断片:
  - `この先へは 行かないほうが いいですよ!`
  - `人気度が いっぺんに 13も下がってしまう`
  - `ゲームクリアに かんけい ありません`
  - `それでも いきますか?`
  - `いちばんきらい`
  - `続いて 2番目`
  - `最後`
- 解釈:
  - Web情報の「嫌いな人の名前を3人入力」と完全に整合。
  - `0x09BF28..0x09BF5A` は名前入力順位プロンプトとして固定してよい。
  - `0x09BE45..0x09BE60` は人気度13低下の警告文。
- Goal影響:
  - Goal 9: イベント仕様書化が前進。
  - Goal 11: 会話/イベント外部データ化候補として確度上昇。

## 未解決文字・制御の推定更新
- `0x083440` 近辺の `1A F2 1A F3 ... 18 FE 18 18` は「半分」「値段」で確定的に読める。
- `0x0830F0` 近辺の `2ばいに ちりょうひを` は、人気度30未満の医者料金2倍文として読める。
- `0x09BF28` 近辺の崩れは、普通の本文崩れではなく、順位入力・名前入力・動的差し込み制御が混ざっているため、静的デコードだけでは完全復元困難。
- `十一屋` と読める文字列は、攻略上の「といちや」と同一機能として扱う。表記ゆれではなく当て字/屋号表記の可能性が高い。
