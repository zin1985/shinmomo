# 新桃太郎伝説 小文字コード F6/F7/F8 見直し

## 結論

`F8` は現行Luaで `ゅ` になっていましたが、実ログとROM内一致から **`F8 = ょ`** に修正するのが妥当です。

根拠は、銀次装備会話の末尾です。

```text
91 F5 9B F8 = い + っ + し + ?
```

画面文は「いっしょだがな！」なので、末尾 `F8` は `ょ` です。

## 確認できた対応

| byte | 修正後 | 根拠 |
|---|---|---|
| `F5` | `っ` | `91 F5 9B F8` = いっしょ、`9B F6 F5 96 BD` = しゃっきん |
| `F6` | `ゃ` | `9B F6 F5 96 BD` = しゃっきん |
| `F8` | `ょ` | `91 F5 9B F8` = いっしょ |
| `F7` | `ゅ` 候補 | 今回のログだけでは確定弱め。暫定で `ゅ` 候補扱い |

## 重要な認識更新

v27で出た

```text
いっしゅだがな
```

は、mode02 decoderの問題ではなく、文字コード表で `F8` を `ゅ` としていたのが原因です。

修正後は、

```text
91 F5 9B F8 = いっしょ
```

になります。
