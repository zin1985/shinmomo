# 新桃太郎伝説 文脈予測語のROM静的照合 v19

## 実施内容

画面文脈から予測した未解決語を、既知文字コードでROM内検索し、辞書・static sequenceへ追加した。

## 完全一致しなかった語

| 語 | 見つからなかったtoken | 解釈 |
|---|---|---|
| 銀次のそうび | `18 50 18 51 A8 9E 92 DF` | 途中に制御/辞書/分岐が挟まる可能性 |
| 銀次は刀 | `18 50 18 51 A9 18 4C` | 同上 |
| 刀も | `18 4C B2` | 同上 |
| そうびを | `9E 92 DF BC` | 同上 |
| 北西 | `18 13 18 11` | 単体の北/西はあるが連続なし |
| 入れません | `02 D7 AE 9D BD` | 入れは02D7、ません側が別処理の可能性 |

## v19の狙い

`src_raw` が複数tokenを含む場合に、

```text
static_hits=銀次の / そうび
context_candidates=銀次のそうび
```

のように、全文固定置換ではなく、ROM内hitの組み合わせとして候補化する。
