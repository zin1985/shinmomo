# 新桃太郎伝説 F6/F7/F8 小文字かな再検証

## 結論

`F8 = ょ` で確定寄りです。

以前の説明で「いっしゅ」と書いたのは誤りです。  
BD98 decoderの問題ではなく、説明側の取り違えでした。

## 確認結果

| byte列 | 読み | ROM出現 |
|---|---|---:|
| `9B F6 F5 96 BD` | しゃっきん | 1 |
| `91 F5 9B F8` | いっしょ | 8 |
| `9B F6` | しゃ | 41 |
| `9B F7` | しゅ候補 | 42 |
| `9B F8` | しょ | 119 |

## 割当

| code | 文字 |
|---|---|
| `F5` | `っ` |
| `F6` | `ゃ` |
| `F7` | `ゅ` |
| `F8` | `ょ` |

## 反映

v28では、MOMO3表に対して以下を明示上書きしています。

```lua
MOMO3["F5"] = "っ"
MOMO3["F6"] = "ゃ"
MOMO3["F7"] = "ゅ"
MOMO3["F8"] = "ょ"
```
