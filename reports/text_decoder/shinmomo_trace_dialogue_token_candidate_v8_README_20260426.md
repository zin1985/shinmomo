# shinmomo dialogue token candidate tracer v8

## v8の目的

v7はかな部分の取得がかなり改善しましたが、以下が不足していました。

- 桃太郎
- 鬼
- 穴
- 言
- おむすびころりんの穴
- おそわれたと
- 言ってただ
- 改行/空白/句読点整理

v8では、実測rawに加えて `corrected=` という補正ビューを出します。

## 重要

`raw_linebuf=` は実測寄りです。  
`corrected=` は推定補正です。  
GitHub資料にはこの2つを分けて記録してください。

## 追加した主な処理

既存:

```text
02A0 = 桃太郎
02AC = オニ
02B0 = きんたん
02B9 = ゆうき
02C0 = おにぎり
02CD = ちから
02CE = あしゅら
```

暫定追加:

```text
029B = 旅の人
```

### 2. 漢字ヒント

```text
1800 = 桃
1801 = 太
1802 = 郎
18DB = 鬼
19EB = 穴
1A1A = 言
```

### 3. 補正ビュー

現在のスクショ/ログに対して強い補正:

```text
さん -> 桃太郎さん
えらこちゃ -> えらいこっちゃ
おむすびこりのがぢににに -> おむすびころりんの穴が 鬼に
おそわた -> おそわれたと
旅の人がてただ -> 旅の人が 言ってただ
```
