# 新桃太郎伝説 未解決文字・熟語補完 v18 静的解析メモ

## 結論

これまでのログで欠けていたのは「かなコードしか取れていない」ためではなく、`$12B2/$12C4` が表示作業窓で、原文の辞書語・漢字トークンがすべては流れてこないため。

v18では `src_raw` に対して静的辞書を直接当て、`static_hits=` を出す。

## 重要な認識

`1850=銀` はノイズではない。`1850 1851=銀次` がROM内に17件存在するため、銀次会話では本文候補として扱う。

## 次の確認方法

v18で同じ会話を取り、以下を見る。

```text
TRACE_DIALOGUE_V18_SOURCE
src_decode=
static_hits=
```

`static_hits` に `銀次`、`刀`、`装備` が出れば、表示窓で欠けた熟語・漢字を原文source側から補完できている。
