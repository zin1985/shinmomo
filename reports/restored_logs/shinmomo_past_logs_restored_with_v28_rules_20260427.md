# 最新ルール(v28相当)で過去ログを復元

※文脈による固定置換ではなく、C0:BD98 bitstream decoder + 小文字かな `F5=っ/F6=ゃ/F7=ゅ/F8=ょ` を適用した復元です。

## 銀次装備/装備確認・着流し説明系

- source: `C8:A7DD` / ROM offset `0x08A7DD`
- decoded symbols: 92

### 復元本文

```text
「桃太郎！ 銀次の そうびを
 ととのえたか?
 銀次は 刀も そうびできる！」
「銀次の そうびは 着流しだ！
 はちまきと わらじは
 桃太郎と いっしょだがな！」
```

## 過去の display_decode 断片への最新ルール適用メモ

| ログ種別 | 過去の断片 | 最新ルールでの扱い |
|---|---|---|
| 北西のおむすびころりんの穴系 | `のおむすこりのはははおむすを なこなとと入れまんよ!ん!」」` | `のおむすこりのはははおむすを なこなとと入れまんよ!ん!」」` / 表示作業窓由来なので欠落あり。v28ではmode02本体から復元するのが本命。 |
| その体で鬼たいじ系 | `その たじ くのかをするな言っももことじ むゃな」` | `その たじ くのかをするな言っももことじ むゃな」` / 表示作業窓由来。体/鬼/無理/桃太郎などは表示stageで落ちやすい。 |
| 銀次装備確認系 | `    そうをををとのたか   はも うびでる!」」` | `    そうをををとのたか   はも うびでる!」」` / v28 mode02 decodeでは『銀次の そうびを ととのえたか？ 銀次は 刀も そうびできる！』に復元可能。 |
| 銀次装備・着流し説明系 | `   のそうび !!!はちまと わじははは いっゅが!」」` | `   のそうび !!!はちまと わじははは いっしょが!」」` / F8=ょ確認後、対応箇所は『いっしょ』。表示stageのみでは『桃太郎』『着流し』が落ちる。 |
