# shinmomo vol016 thread handover package

This folder is ready to be committed into `zin1985/shinmomo` or merged manually.

## Contents

- `handover/` final thread handover, goal progress, NPC patch feasibility, graphics next steps
- `graphics/` field/battle/static/mapchip output files generated in this thread
- `data/runtime_logs/` runtime CSV/bin logs including `vram.bin`
- `data/legacy_tables/` older CSV handover tables used as references
- `docs/legacy_handover/` older markdown handovers
- `docs/disassembly/` disassembly text used in analysis
- `tools/lua/` Lua loggers/specs
- `tools/python/` and analysis scripts inside graphics folders
- `rom/README_ROM_NOT_INCLUDED.md` explains why ROM is not included
- `manifest/files_manifest.csv` file listing with checksums

## Commit

```bash
git add .
git commit -m "Add vol016 graphics mapchip and NPC display handover package"
```

## Important

ROM binaries are intentionally excluded. Keep them local and never commit them.
