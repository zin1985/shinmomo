# 新桃太郎伝説 現時点カラーキャラクター候補出力 2026-05-02

## 重要な注意
このパッケージは **現時点で手元にある VRAM dump / DA:3800 decoded records / visible object CSV から作った仮カラー出力**です。
`cgram.bin` / `oam.bin` が手元に無いため、ゲーム実機の正確な色・正確なOAM組み立てではありません。

## 主な画像
- `graphics/characters/current_vram_all_4bpp_pseudocolor_tilesheet.png`
  - `vram.bin` 全体を 4bpp として仮カラー表示したタイルシート。
- `graphics/characters/current_vram_character_candidate_groups_top64_pseudocolor.png`
  - VRAM 4bppを 16x24 / 16x32 / 32x32 の連続タイル塊として切り、キャラクター候補っぽいものを上位64件で並べたもの。
- `graphics/characters/da3800_stream08_22_colored_character_glyph_candidates.png`
  - DA:3800 stream 8..22 の decoded records を仮カラー化した候補一覧。
- `graphics/characters/da3800_stream01_07_colored_display_candidates.png`
  - DA:3800 stream 1..7 の表示資材候補を仮カラー化したもの。
- `graphics/characters/battle_visible_objects_table_colorized_reference.png`
  - visible object 一覧表の参照用カラー化。

## CSV
- `data/csv/current_character_vram_candidate_groups.csv`
  - 候補chunkのVRAM byte offset、shape、score。

## 次に必要なログ
正確なカラーキャラクター画像にするには、同一frameの以下が必要です。

- `vram.bin`
- `cgram.bin`
- `oam.bin`
- `visible_objects.csv`

OAMに tile number / palette / hflip / vflip が入るため、キャラの正しい組み立ては `oam.bin` が決定打です。
