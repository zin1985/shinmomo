# 新桃太郎伝説 graphics full capture v2（2026-05-02）

## 目的
グラフィック再現に必要な情報を、Snes9x/BizHawk Lua から可能な限りまとめて出力するためのパッケージです。

## 収集するもの
- VRAM: `vram.bin`
- CGRAM: `cgram.bin`, `cgram_palette.csv`
- OAM: `oam.bin`, `oam_sprites.csv`, `oam_visible.csv`
- PPU write mirror: `ppu_regs_capture.csv`, `ppu_write_log.csv`
- DMA trigger: `dma_trigger_log.csv`
- BG tilemap: `bg1_tilemap_decoded.csv` ... `bg4_tilemap_decoded.csv`
- 全tilemap page総当たり: `all_tilemap_pages_decoded.csv`
- mapchip候補: `bg*_metatile_2x2.csv`
- WRAM候補: object chain / display stack / DMA queue / staging ranges
- preview: `tiles_*bpp_sheet.ppm`, `bg*_preview.ppm`

## 使い方
1. ROM起動直後、または目的画面に入る前に `tools/lua/shinmomo_graphics_full_capture_v2_snes9x_bizhawk_20260502.lua` を実行。
2. フィールド、町、屋内、戦闘、会話ウィンドウあり/なしへ移動。
3. `shinmomo_graphics_full_capture_out/frame_xxxxxx/` にsnapshotが作られます。
4. 正確カラー復元では `vram.bin + cgram.bin + bg*_tilemap_decoded.csv` を使います。
5. キャラ復元では `vram.bin + cgram.bin + oam_sprites.csv` を使います。

## 注意
PPUレジスタの多くはwrite-onlyなので、Lua開始後のwriteをmirrorして記録します。最も正確に取るには、ROM起動直後からLuaを実行してください。
