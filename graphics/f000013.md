# 新桃太郎伝説 グラフィック完全復元用 Lua v2 引継ぎ

## 追加ファイル
`tools/lua/shinmomo_graphics_full_capture_v2_snes9x_bizhawk_20260502.lua`

## 目的
グラフィック復元に必要な情報を1回のLua運用で集める。

## 主な出力
- VRAM/CGRAM/OAM raw dump
- CGRAM palette CSV
- OAM decoded CSV
- BG1〜BG4 tilemap decoded CSV
- all tilemap pages decoded CSV
- 2x2 metatile候補CSV
- PPU write mirror
- DMA trigger log
- WRAM object slots / active chain guess
- display stack / DMA queue / staging range dump

## 復元の基本
- BG/mapchip: `vram.bin + cgram.bin + bg*_tilemap_decoded.csv + ppu_regs_capture.csv`
- character/sprite: `vram.bin + cgram.bin + oam_sprites.csv + OBSEL`
- DMA/転送元追跡: `dma_trigger_log.csv + vram_write_log.csv + wram_dma_queue_8000_0800.bin`

## 注意
PPU register はwrite-onlyが多いため、Lua開始後のmirror値である。ROM起動直後から実行するほど正確。
