# Graphics restoration pipeline 20260502

## Layer split

```text
BG/mapchip:
  ROM asset candidate -> decompressed VRAM CHR -> BG tilemap -> CGRAM -> screen

Object/OAM:
  logical actor -> visible object linked list -> OAM mirror -> sprite DMA -> screen
```

## Runtime priority
- VRAM BG tilemap hash
- CGRAM hash / dump
- $0A61 active linked list
- $0AA3 sort/depth key
- OAM visible entries
- map id / scroll candidates

## Success criteria
- Field: 16x16 metatile contact sheet が安定して出る
- Town: 町専用chip候補が出る
- Battle: BG backdrop / UI glyph / OAM sprite を分離できる
